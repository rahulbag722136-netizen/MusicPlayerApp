# 🎵 Music Player App (Core Java)

A simple console-based Music Player application built using Core Java.

## 📌 Features
- View Music Library
- Play Song
- Create Playlist
- Shuffle Playlist
- Volume Control
- Repeat Mode

## 🛠 Technologies Used
- Core Java
- OOP Concepts
- Collections Framework

## ▶ How to Run

1. Open terminal inside project folder
2. Compile all files:

   javac *.java

3. Run the program:

   java MusicApp
